# 视频音频上传与编辑功能 - 需求讨论方案

## 一、需求概述

创建一个支持视频和音频文件上传、预览及剪辑的 Vue 3 应用。

**目标用户**：需要快速剪辑音视频内容的普通用户

**核心价值**：无需安装专业软件，在浏览器中完成简单的音视频剪辑任务

---

## 三、技术实施方案

### 3.1 技术选型

| 模块 | 技术方案 | 说明 |
|------|----------|------|
| **前端框架** | Vue 3 (Composition API) | 项目已采用 |
| **构建工具** | Vite | 项目已采用 |
| **UI 样式** | 原生 CSS / 可选 Tailwind CSS | 保持轻量，按需引入 |

| **媒体剪辑** | `ffmpeg.wasm` | 浏览器端剪辑，无需后端 |
| **拖拽交互** | Pointer Events API | 兼容鼠标和触摸 |

## 五、核心功能实现方案

### 5.1 上传功能

**实现步骤：**

1. 使用 `<input type="file" accept="video/*,audio/*" multiple>` 
2. 通过 `URL.createObjectURL()` 创建本地预览地址
3. 维护上传文件列表状态（响应式数组）
4. 添加文件类型验证和大小限制（100MB）
5. 支持拖拽上传（可选）

**状态结构：**

```javascript
const uploadedFiles = ref([
  {
    id: 'unique-id',
    file: File,              // 原始 File 对象
    url: 'blob:...',         // 预览地址
    type: 'video' | 'audio', // 媒体类型
    name: 'filename.mp4',
    size: 12345678,
    duration: 0,             // 媒体时长（加载后）
    createdAt: Date
  }
])
```

### 5.2 视频编辑器

| 功能点 | 实现方案 |
|--------|----------|
| **播放器** | `<video>` 标签 + 自定义控制栏（播放/暂停、进度条、音量、全屏） |
| **缩略图生成** | 使用 Canvas 每隔 N 秒抽取一帧，生成缩略图序列（建议 10-20 张） |
| **时间轴** | Flex 布局横向排列缩略图，支持横向滚动 |
| **拖拽手柄** | 左右两个绝对定位的拖拽块，使用 Pointer Events 处理 |
| **高亮区域** | 两个手柄之间的背景色高亮（半透明色块） |
| **时长限制** | 拖拽时实时计算，超出 1-25s 范围时限制移动并提示 |
| **剪辑输出** | 使用 ffmpeg.wasm 在浏览器端裁剪视频，输出 blob URL |

**缩略图生成伪代码：**

```javascript
async function generateThumbnails(videoFile, count = 15) {
  const video = document.createElement('video')
  video.src = URL.createObjectURL(videoFile)
  await video.load()
  
  const thumbnails = []
  const interval = video.duration / count
  
  for (let i = 0; i < count; i++) {
    video.currentTime = i * interval
    await new Promise(r => video.onseeked = r)
    
    const canvas = document.createElement('canvas')
    canvas.width = 120
    canvas.height = 68
    const ctx = canvas.getContext('2d')
    ctx.drawImage(video, 0, 0, canvas.width, canvas.height)
    
    thumbnails.push(canvas.toDataURL())
  }
  
  return thumbnails
}
```

### 5.4 拖拽选择器（通用组件）

**核心逻辑：**

```javascript
// ClipRangeSelector.vue 核心逻辑
let isDragging = false
let activeHandle = null // 'left' | 'right'
let startX = 0
let startRange = { start: 0, end: 0 }

function onPointerDown(e, handle) {
  isDragging = true
  activeHandle = handle
  startX = e.clientX
  startRange = { ...clipRange.value }
}

function onPointerMove(e) {
  if (!isDragging) return
  
  const deltaX = e.clientX - startX
  const deltaSeconds = pixelsToSeconds(deltaX)
  
  if (activeHandle === 'left') {
    clipRange.value.start = clamp(startRange.start + deltaSeconds, 0, clipRange.value.end - MIN_DURATION)
  } else {
    clipRange.value.end = clamp(startRange.end + deltaSeconds, clipRange.value.start + MIN_DURATION, duration)
  }
  
  validateDuration() // 检查是否在 1-25s 范围内
}

function onPointerUp() {
  isDragging = false
  activeHandle = null
}
```

**约束条件：**

- 左手柄不能超过右手柄
- 右手柄不能超过左手柄
- 选区时长必须在 1-25s 范围内
- 手柄不能超出时间轴边界

### 5.5 时间格式化

```javascript
// useTimeFormat.js
export function formatTime(seconds) {
  const mins = Math.floor(seconds / 60)
  const secs = Math.floor(seconds % 60)
  const ms = Math.floor((seconds % 1) * 10)
  
  return `${String(mins).padStart(2, '0')}:${String(secs).padStart(2, '0')}.${ms}`
}

// 示例：formatTime(125.5) → "02:05.5"
```

### 6.2 关键 Composables

**useMediaClip.js**

```javascript
export function useMediaClip() {
  const clip = async (file, startTime, endTime, type) => {
    const ffmpeg = await loadFFmpeg()
    
    // 准备输入文件
    ffmpeg.writeFile('input', await fetchFile(file))
    
    // 执行剪辑命令
    await ffmpeg.exec([
      '-i', 'input',
      '-ss', startTime.toString(),
      '-t', (endTime - startTime).toString(),
      '-c', 'copy',
      'output' + getFileExtension(type)
    ])
    
    // 读取输出
    const output = await ffmpeg.readFile('output' + getFileExtension(type))
    const blob = new Blob([output], { type: getMimeType(type) })
    
    return URL.createObjectURL(blob)
  }
  
  return { clip }
}
```

## 八、潜在风险与解决方案

| 风险 | 影响程度 | 解决方案 |
|------|----------|----------|
| **ffmpeg.wasm 体积大 (~25MB)** | 高 - 首次加载慢 | 按需加载、显示加载进度条、考虑 CDN |
| **视频缩略图生成慢** | 中 - 大视频卡顿 | Web Worker 后台处理、限制缩略图数量、降低分辨率 |
| **移动端兼容性** | 中 - 拖拽不灵敏 | 使用 Pointer Events、增加触摸区域、测试主流机型 |
| **大文件处理内存溢出** | 高 - 浏览器崩溃 | 限制文件大小 (100MB)、分块处理、给出友好提示 |
| **浏览器格式兼容性** | 中 - Safari 不支持某些格式 | 提供格式支持提示、降级方案、测试主流浏览器 |
| **ffmpeg.wasm 多线程支持** | 低 - 部分浏览器不支持 | 检测支持性，降级到单线程版本 |
